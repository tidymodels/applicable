# Binary QSAR Data

Binary QSAR Data

## Value

- binary_tr,binary_ukn:

  data frame frames with 67 columns

## Details

These data are from two different sources on quantitative
structure-activity relationship (QSAR) modeling and contain 67
predictors that are either 0 or 1. The training set contains 4,330
samples and there are five unknown samples (both from the `Mutagen` data
in the `QSARdata` package).
