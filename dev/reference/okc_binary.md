# OkCupid Binary Predictors

OkCupid Binary Predictors

## Source

Kim (2015), "OkCupid Data for Introductory Statistics and Data Science
Courses", *Journal of Statistics Education*, Volume 23, Number 2.
[doi:10.1080/10691898.2015.11889737](https://doi.org/10.1080/10691898.2015.11889737)

Kuhn and Johnson (2020), *Feature Engineering and Selection*, Chapman
and Hall/CRC . <https://bookdown.org/max/FES/> and
<https://github.com/topepo/FES>

## Value

- okc_binary_train,okc_binary_test:

  data frame frames with 61 columns

## Details

Data originally from Kim (2015) includes a training and test set
consistent with Kuhn and Johnson (2020). Predictors include ethnicity
indicators and a set of keywords derived from text essay data.
